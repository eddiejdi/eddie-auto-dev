## Technical Details

### Pre-Commit Hook Enhancement

**What Changed**: Updated `.githooks/pre-commit` to allow **deletion** of `.env` files while still blocking **creation/modification** of files containing secrets.

**Before**:
```bash
if git diff --cached --name-only | grep -E "^\.env$|^\.env\.[a-z]"
```
→ Blocked both ADD/MODIFY/DELETE operations

**After**:
```bash
ADDED_OR_MODIFIED_ENV=$(git diff --cached --diff-filter=AM --name-only | grep -E "^\.env$|^\.env\.[a-z]")
```
→ Only blocks ADD/MODIFY; allows DELETE operations

### Deployment Script Verification

All deploy scripts create their `*.env` files **on-demand** — they do not depend on pre-existing files:

1. **deploy_email.sh** (line ~148)

   EOF
   ```

2. **deploy_mailu.py**

   ```

3. **deploy_simple_mail.py**
   ```bash

   ```
