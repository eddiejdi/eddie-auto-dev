🔧 TROUBLESHOOTING
═══════════════════════════════════════════════════════════════════════════

If GPU not detected in Ollama:
   1. Check dmesg for errors: sudo dmesg | grep -i nvidia
   2. Unset CUDA_VISIBLE_DEVICES: unset CUDA_VISIBLE_DEVICES
   3. Restart Ollama: sudo systemctl restart ollama
   4. Verify: nvidia-smi

If inference is slow (using CPU):
   • Normal for 14B models with GTX 1050 (2GB VRAM)
   • Use smaller models (7B) for faster performance
   • Check nvidia-smi during inference for GPU utilization

If Ollama crashes after GPU config:
   • Revert force-cuda.conf temporarily
   • Check logs: journalctl -u ollama -f
   • Verify CUDA toolkitinstalled: which cuda-smi
