🪟 Instalação no Windows

1
Baixe e instale o WireGuard

Instale o cliente oficial para Windows (64-bit). Requer Windows 10/11.

⬇ Baixar WireGuard para Windows
2
Baixe o arquivo de configuração

Faça o download do seu perfil VPN pessoal.

⬇ Baixar rpa4all-vpn.conf
3
Importe no WireGuard

Abra o WireGuard → clique em "Importar túnel(s) de arquivo..." → selecione o arquivo .conf baixado.

4
Ative o túnel

Clique em "Activar". O status muda para Ativo quando a conexão for estabelecida.

🐧 Instalação no Linux

ℹ️ Você precisa estar logado neste portal para baixar o arquivo de configuração. O script abaixo só funciona após o download manual do .conf.

Passo 1 — Instale o WireGuard

Escolha o comando para sua distribuição:

Debian / Ubuntu

sudo apt update && sudo apt install -y wireguard

Fedora / RHEL / Rocky

sudo dnf install -y wireguard-tools

Arch Linux

sudo pacman -S wireguard-tools

Passo 2 — Baixe a configuração

Faça o download do perfil VPN:

⬇ Baixar wg0.conf

Passo 3 — Instale o perfil e ative

Depois de baixar, execute (substituindo pelo caminho do arquivo):

sudo cp ~/Downloads/wg0.conf /etc/wireguard/wg0.conf sudo chmod 600 /etc/wireguard/wg0.conf sudo systemctl enable --now wg-quick@wg0

Controle do túnel

sudo wg-quick up wg0 # conectar sudo wg-quick down wg0 # desconectar sudo wg show # status

🤖 Instalação no Android

1
Instale o app WireGuard

Disponível na Play Store gratuitamente.

▶ Play Store 📦 F-Droid
2
Escaneie o QR Code ou baixe o arquivo

No WireGuard → "+" → "Escanear QR code", ou baixe o .conf e importe.

Se preferir arquivo:

⬇ Baixar .conf 🔍 Abrir QR ampliado
3
Ative o túnel

Toque no toggle do perfil importado. Aceite a criação da VPN quando solicitado pelo Android.

📱 DNS Privado (Private DNS): Para bloquear anúncios no Android via Pi-hole mesmo em 4G/5G, configure em Configurações → Rede → DNS Privado o endereço auth.rpa4all.com (requer VPN ativa para funcionar).

🍎 Instalação no iPhone / iPad

1
Instale o app WireGuard

Gratuito na App Store da Apple.

🍎 App Store — WireGuard
2
Escaneie o QR Code ou baixe o arquivo

No WireGuard → "+" → "Criar com QR code", ou com o botão abaixo.

Alternativa — arquivo:

⬇ Baixar .conf 🔍 Ampliar QR
3
Ative o túnel

Toque no toggle do perfil. O iOS pedirá permissão para adicionar configuração de VPN — toque em Permitir.

🔒 O iOS pode exigir autenticação biométrica (Face ID / Touch ID) para ativar a VPN. Isso é normal e esperado.
