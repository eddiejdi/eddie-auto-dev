🪟 Instalação Automática — Windows 10/11

1
Baixe o instalador automático

O instalador verifica se o WireGuard está instalado, instala via winget caso necessário, configura e ativa o túnel. Basta dar duplo-clique!

2
Execute com duplo-clique

Dê duplo-clique no arquivo install-vpn.bat. O Windows pedirá permissão de Administrador (UAC) automaticamente.

3
Pronto!

O túnel VPN será ativado automaticamente. Ícone WireGuard aparece na bandeja do sistema.

⚠️O Windows pedirá permissão de Administrador (UAC) automaticamente. Se o SmartScreen bloquear, clique em "Mais informações" → "Executar assim mesmo".

🐧 Instalação Automática — Linux

1
Baixe o instalador automático

Detecta distro (apt/dnf/pacman/zypper), instala WireGuard, configura e ativa o túnel. Basta um comando.

2
Execute no terminal
sudo bash ~/Downloads/install-vpn.sh
3
Pronto!

O túnel é ativado e habilitado no boot. Para controlar:

sudo wg-quick up wg0 # conectar sudo wg-quick down wg0 # desconectar sudo wg show # status sudo systemctl status wg-quick@wg0

Manual — escolha sua distro

Debian / Ubuntu

sudo apt update && sudo apt install -y wireguard

Fedora / RHEL

sudo dnf install -y wireguard-tools

Arch

sudo pacman -S wireguard-tools
