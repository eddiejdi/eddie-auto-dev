🆘 Solução de Problemas

❌ "Não consigo conectar ao homelab"

  • Verifique: Você está na mesma rede LAN (192.168.15.0/24)?
  • Se remoto: Use WireGuard VPN (solicite ao admin)
  • Teste: ping 192.168.15.2

❌ "Serviço X está offline"

  1. Verifique o status via Grafana (painel "System Status")
  2. SSH no homelab: systemctl status nome-servico
  3. Veja logs: journalctl -u nome-servico -f
  4. Se não responde: contate o administrador

❌ "GPU está lenta/congelada"

  • Verifique em Grafana → "GPU Utilization"
  • SSH: nvidia-smi mostra uso em tempo real
  • Se travar: o sistema reconstrói automaticamente em 5 min

❌ "Não consigo fazer login"

  • Verifique: você solicitou criação de conta em Authentik?
  • Resete senha: Link de recuperação
  • Contate o admin se a conta não existe
